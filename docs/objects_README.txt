--------------------------------------------------
Circle: 
  (x - x0) ^ 2 + (y - y0) ^ 2 = r ^ 2
--------------------------------------------------

--------------------------------------------------
Rectangle:
  |x - x0| <= Lx / 2
  |y - y0| <= Ly / 2
--------------------------------------------------

--------------------------------------------------
Mountain:
  y(x) = y0 - sqrt(lambda ^ 2 * (x - x0) ^ 2 + h ^ 2)
--------------------------------------------------

--------------------------------------------------
Airfoil:
  Defined only when x0 <= x <= x0 + 1

  Let f(x) := a * sqrt(x - x0) + b * (x - x0) + c * (x - x0) ^ 2 + d * (x - x0) ^ 3 + e * (x - x0) ^ 4

  (top surface):
    y_t(x) = y0 + f(x)
  (bottom surface):
    y_b(x) = y0 - lambda * f(x)
--------------------------------------------------
